1. PROJECT DESCRIPTION
VOYAGER AI is an AI-powered travel planner that creates personalized travel itineraries based on destination, budget, trip duration, and preferences. It provides daily plans, hotel suggestions, activities, and budget estimates.

2. REQUIREMENTS
- Python 3.8+
- Flask
- SQLAlchemy
- Anthropic API Key
- Internet connection
- Modern browser (Chrome/Edge/Firefox)

3. HOW TO RUN PROJECT
1. Download project files
2. Install dependencies:
   pip install -r requirements.txt
3. Add API keys in .env file
4. Run project:
   python app.py
5. Open:
   http://localhost:5000

4. HOW IT WORKS (TECH FLOW)
User Input → Form Validation → AI Prompt Generation → Claude API → Travel Plan Creation → Save to Database → Display Itinerary

5. FEATURES
- AI itinerary generation
- Hotel & restaurant suggestions
- Budget planning
- Trip saving & sharing
- User login system
- Responsive design

6. COMMON ISSUES & FIXES
- API key error → Check .env file
- Port busy → Change port
- Database error → Reinitialize DB
- Slow response → Check internet/API

7. SUPPORTED BROWSERS
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

8. TIPS FOR BEST RESULTS
- Enter clear preferences
- Use realistic budgets
- Choose correct trip type
- Review itinerary before travel
