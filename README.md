# Abstract

AI Travel Planner is a web-based application that helps users plan trips quickly and efficiently. By using Artificial Intelligence, the system generates personalized travel itineraries based on inputs such as destination, budget, duration, and interests. Instead of spending hours researching places and activities, users receive a customized travel plan within seconds. The project demonstrates how AI can simplify travel planning and improve the overall travel experience.

---

# Introduction

Planning a trip can be exciting but also time-consuming. Travelers often need to search multiple websites for destinations, attractions, accommodations, and schedules. The AI Travel Planner aims to solve this problem by generating personalized travel recommendations automatically. The goal is to make travel planning easier, faster, and more convenient for users.

---

# Literature Review

Many travel platforms provide recommendations, but they often require significant user effort. Recent advancements in Artificial Intelligence and Large Language Models have made it possible to generate personalized suggestions through natural language interactions. This project applies these technologies to create smart travel itineraries.

---

# Methodology

The user enters travel details such as destination, budget, trip duration, and preferences. The application processes this information and sends it to an AI model. Based on the provided inputs, the AI generates a customized itinerary including places to visit, activities, travel tips, and estimated expenses. The results are displayed through a simple and interactive interface.

# Results and Discussion

The application successfully generates personalized travel plans within a few seconds. Users receive destination-specific recommendations, daily schedules, and useful travel suggestions. The system reduces planning time and provides a smooth user experience.

---

# Limitations

* Requires internet connectivity.
* Recommendations depend on user inputs.
* Does not currently support direct flight or hotel booking.
* AI responses may occasionally vary in accuracy.
